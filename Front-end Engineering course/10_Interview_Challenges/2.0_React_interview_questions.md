# React Interview Questions

---

## English

1. What is the difference between the Virtual DOM and the real DOM?
2. Is the Virtual DOM the same as the Shadow DOM?
3. What is JSX?
4. What is the difference between an element and a component?
5. Can you write React without JSX?
6. How do you pass a value from parent to child? What about from child to parent?
7. What is prop drilling?
8. Can you modify props?
9. What is the difference between props and state?
10. What is the difference between state in a class component versus state in a function component?
11. What is the component lifecycle?
12. How do you handle lifecycle behavior in function components?
13. What parameters does useEffect take?
14. When does useEffect run?
15. What is the return value of useEffect?
16. What is the difference between refs and state variables?
17. When is it best to use refs?
18. What is the proper way to update a ref in a function component?
19. What is the difference between the Context API and prop drilling?
20. When should you avoid using the Context API?
21. What is a Fragment?
22. When should you use a class-based component versus a function component?
23. What is a Higher-Order Component (HOC)?
24. What is a portal?
25. What are controlled and uncontrolled components?
