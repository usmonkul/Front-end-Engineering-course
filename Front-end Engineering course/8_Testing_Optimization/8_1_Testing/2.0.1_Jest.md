## Jest bo‘yicha qisqa qo‘llanma

Bu qisqa darslikda Jest’ni yangi boshlovchilar uchun eng kerakli qismlarini ko‘ramiz. Maqsad — oddiy unit testlarni tez va to‘g‘ri yozishni o‘rganish.

### 1. Jest nima?

Jest — JavaScript uchun test runner. U testlarni ishga tushiradi, natijani ko‘rsatadi va mock qilish imkonini beradi.

### 2. O‘rnatish

Loyihada `npm` ishlatilsa:

```bash
npm install --save-dev jest
```

`package.json`ga skript qo‘shiladi:

```json
{
  "scripts": {
    "test": "jest"
  }
}
```

Testlarni ishga tushirish:

```bash
npm test
```

### 3. Test fayl nomi

Jest avtomatik topishi uchun:

- `sum.test.js`
- `sum.spec.js`

Odatda test fayl kod yonida bo‘ladi.

### 4. Eng oddiy test

```javascript
function sum(a, b) {
  return a + b;
}

test("sum 2 va 3 ni 5 qaytaradi", () => {
  expect(sum(2, 3)).toBe(5);
});
```

### 5. Asosiy matcherlardan foydalanish

- `toBe()` — aniq qiymat
- `toEqual()` — obyekt yoki array
- `toBeTruthy()` — true bo‘lishi
- `toBeFalsy()` — false bo‘lishi
- `toContain()` — ichida borligini tekshiradi

Misollar:

```javascript
// toBe()
expect(2 + 3).toBe(5);

// toEqual()
expect({ name: "Ali" }).toEqual({ name: "Ali" });

// toBeTruthy()
expect("text").toBeTruthy();

// toBeFalsy()
expect(0).toBeFalsy();

// toContain()
expect(["red", "green", "blue"]).toContain("green");
```

### 6. Grouplash va tartib

Testlarni guruhlash uchun:

```javascript
describe("Math utils", () => {
  test("sum ishlaydi", () => {
    expect(sum(1, 2)).toBe(3);
  });
});
```

### 7. Async test (oddiy ko‘rinish)

```javascript
test("async funksiya qiymat qaytaradi", async () => {
  const result = await getData();
  expect(result).toBe("ok");
});
```

### 8. Mock qilish (oddiy tushuncha)

API yoki modulni soxta qilib tekshirsa bo‘ladi:

```javascript
jest.mock("./api");
```

Junior uchun yetarlisi — real API chaqirmaslik va testni tez ishlatish.

### 9. Eng ko‘p uchraydigan xatolar

- Testlar bir-biriga ta’sir qiladi (shared state)
- Vaqtga bog‘liq testlar yiqiladi (flaky timers)

**Yechim**: har testda alohida data ishlating va kerak bo‘lsa fake timers qo‘llang.
