CSS `position` — static, relative, absolute, fixed

Ushbu sahifa — dars o‘tish uchun tayyor template. Har bir bo‘limda: qisqa nazariya + demo + ko‘p uchraydigan xatolar va mini-vazifa.

1) `position: static` (default)

default

Static — elementning standart holati. Element normal oqimda (normal flow) joylashadi.

  • `top/right/bottom/left` ishlamaydi (ta’sir qilmaydi).
  • Element boshqa elementlarni “itarmaydi” yoki ustiga chiqmaydi — hammasi oddiy oqimda bo‘ladi.
O‘qituvchi eslatma: talabalar ko‘pincha `top: 20px` yozib “nega yurmayapti?” deydi. Sababi: element `position: static`.

DEMO: `top/left` berilgan, lekin static bo‘lgani uchun o‘zgarmaydi

A
B
C
.box.b {
  position: static;
  top: 20px;   /* ishlamaydi */
  left: 20px;  /* ishlamaydi */
}

2) `position: relative`

offset + anchor

Relative — element normal oqimda qoladi, lekin `top/left/right/bottom` bilan o‘z joyidan “siljiydi”.

  • `top/left/...` ishlaydi.
  • Elementning “asl joyi” saqlanib qoladi (bo‘sh joy qoladi).
  • Ko‘pincha `absolute` elementlar uchun konteyner qilishda ishlatiladi.

DEMO: B blok `relative` bo‘lib pastga/chapga siljiydi

A
B
C
.box.b {
  position: relative;
  top: 18px;
  left: 18px;
}
Mini-vazifa: `top`ni `-18px` qilib ko‘ring. B qaysi tomonga ketadi?

3) `position: absolute`

out of flow

Absolute — element normal oqimdan chiqadi (out of flow) va eng yaqin position berilgan ancestorga nisbatan joylashadi.

  • Ko‘pincha badge, icon, dropdown, tooltip, modal elementlar uchun.
  • Agar parent’da `position: relative` (yoki `absolute/fixed/sticky`) bo‘lmasa, u holda viewportga (sahifaga) nisbatan joylashishi mumkin.
Eng ko‘p xato: parent’ga `position: relative` berilmasa, `absolute` element “kutilmagan joy”ga ketadi.

DEMO: B blok sahna (stage) ichida yuqori o‘ngga “yopishadi”

A
C
B
.stage { position: relative; }

.box.b {
  position: absolute;
  top: 16px;
  right: 16px;
}
Mini-vazifa: `right: 16px` o‘rniga `left: 16px` qilib ko‘ring.

4) `position: fixed`

viewport

Fixed — element viewportga (brauzer oynasiga) nisbatan joylashadi. Sahifani scroll qilsangiz ham joyida turadi.

  • Ko‘pincha sticky CTA, “back to top” button, chat widget, fixed navbar uchun.
  • Ko‘pincha `z-index` kerak bo‘ladi (ustida turishi uchun).

DEMO: B blok sahifaning past-o‘ng burchagiga fixed

Scroll qilish uchun joy (pastga tushing)

Pastga scroll qiling — B blok joyida qoladi. 

.box.b {
  position: fixed;
  bottom: 18px;
  right: 18px;
  z-index: 10;
}
Mini-vazifa: `bottom`ni `80px` qiling. Element qayerga ko‘chdi?

Cheat sheet (dars yakuni)

quick recap
  • static: default, `top/left` ishlamaydi.
  • relative: oqimda qoladi, o‘z joyidan siljiydi, ko‘pincha absolute uchun konteyner.
  • absolute: oqimdan chiqadi, eng yaqin position’li parent’ga nisbatan joylashadi.
  • fixed: viewportga nisbatan, scroll qilsangiz ham joyida qoladi.
Uyga vazifa g‘oya: 1 ta card yarating. Card ichida “SALE” badge bo‘lsin (absolute). Past-o‘ngda “Help” tugmasi bo‘lsin (fixed).
